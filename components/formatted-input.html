# Formatted Input

import { formattedInput } from 'vigour';
// formattedInput.name === 'vigour-formatted-input'

Formatted Input only accept number. It can be used as a credit card number input field.

Formatted Input 只能接受数字输入。它可以用作信用卡号码的输入框。
