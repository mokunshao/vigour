# onToggle & onClick

The onToggle and onClick attributes accept a function as an argument.

onToggle 和 onClick 属性接受一个函数作为参数。

  :data="json"
  :onToggle="onToggle"
  :onClick="onClick"
></vigour-tree>

    console.log('onToggle', data);
  },
  onClick(data) {
    console.log('onClick', data);

},
